------------------------------- README ----------------------------------
Language used : Python 3.6.2

Libraries used : 
1) pandas
2) matplotlib
3) sklearn

Dataset used: Balance-Scale dataset.

How to compile:
1. Install all the library packages used.
2. The folder contains a python notebook ‘CompareClassifiers.ipynb’. It can be opened using the Jupyter notebook.
3. Run the python notebook. (Jupyter recommended)
4. View log file and report for results and analysis.
 
Classifiers compared:
1. Decision tree
2. Perceptron
3. Neural net
4. Deep Learning
5. SVM
6. Naive Bayes 
7. Logistic Regression
8. k-nearest neighbor
9. Bagging
10. Random forests
11. Adaboost
12. Gradient boosting
